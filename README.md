# FlutterHelloApp
# 1. 引用
hello flutter 练手项目
https://flutterchina.club/get-started/codelab/

# 2. 功能说明
随机生成英文公司名称列表，支持收藏/取消收藏 item。
可以跳转收藏页面，查看已收藏的 item。
应用不支持持久缓存，重新打开 APP 后重新生成数据，收藏数据清空。
